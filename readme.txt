An AI-powered content studio for WordPress featuring grammar checking, AI text humanization, and AI detection bypass tools.

== Description ==

TextCraft AI is a powerful WordPress plugin that enhances your content with cutting-edge AI technology. It provides three essential tools for content creators:

1. **Grammar Checking**: Automatically detect and correct grammatical errors in your text.
2. **AI Text Humanization**: Transform AI-generated content to sound more natural and human-like.
3. **AI Detection Bypass**: Process text to bypass AI content detection systems.

= Key Features =

* Real-time grammar checking with detailed error analysis
* AI text humanization to make content sound more natural
* AI detection bypass to avoid content flagging
* Clean, intuitive interface
* Fast processing using external APIs
* Responsive design that works on all devices

= How It Works =

1. Install and activate the plugin
2. Navigate to the TextCraft AI menu in your WordPress admin
3. Enter your text in the provided text area
4. Choose one of the three processing options:
   - Check Grammar: Identifies and corrects grammatical errors
   - Humanize AI Text: Makes AI-generated content sound more natural
   - Bypass AI Detection: Processes text to avoid AI content detection
5. View the processed results and use as needed

= Perfect For =

* Content creators who use AI writing tools
* Bloggers and writers wanting to improve their content
* SEO professionals optimizing text for search engines
* Educators and students enhancing their writing
* Anyone looking to improve their written communication

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/textcraft-ai` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the TextCraft AI menu in the WordPress admin to access the tools

== Frequently Asked Questions ==

= Do I need any special technical skills to use this plugin? =

No, TextCraft AI is designed to be user-friendly and requires no technical skills. Just install, activate, and start using the tools.

= Does this plugin store my content? =

No, your content is only sent to the AI APIs for processing and is not stored permanently.

= Is there a limit to how much text I can process? =

The plugin itself doesn't impose any limits, but the external APIs may have their own limitations.

= Can I use this plugin with any WordPress theme? =

Yes, TextCraft AI works with any WordPress theme.
